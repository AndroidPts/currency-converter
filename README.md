## Конвертер валют с кэшированием

Консольное приложение для конвертации валют с сохранением курсов в кэш и ведением истории запросов.  
Практическая работа №2 по дисциплине «Инструментальные средства разработки ПО».

## Цель работы

Освоение командной разработки с использованием Git и GitHub: создание веток, проведение Code Review, слияние Pull Request'ов.

## Функциональность

- 🚀 Автоматическое получение актуальных курсов валют через API
- ⚡ Умное кэширование данных на 1 час (файл `currency_cache.json`)
- 📚 Сохранение последних 5 конвертаций в истории
- 🔄 Поддержка конвертации через рубль для всех мировых валют
- 🛡️ Обработка ошибок сети и неверного ввода пользователя

## Автоматическая проверка кода (CI/CD)

При каждом push и pull request в ветки `main` и `develop` автоматически запускаются:
- Проверка форматирования (`black --check .`)
- Проверка стиля кода (`flake8 src/`)
- Запуск юнит-тестов (`pytest tests/ -v`)

Статус проверок отображается в badge вверху страницы и на вкладке Actions.

# Разработка
**Установка зависимостей**
Для работы программы и запуска тестов необходимо установить библиотеки.

```bash
pip install -r requirements.txt
```

## Запуск тестов
**Юнит-тесты написаны с использованием pytest. Для запуска выполните:**

```bash
pytest tests/ -v
```

## Проверка стиля кода
**Для проверки кода на соответствие стандартам PEP8 используется линтер flake8.**

```bash
flake8 src/
```

## Форматирование кода
**Для автоматического приведения кода к единому стилю используется black.**

```bash
black src/
```
